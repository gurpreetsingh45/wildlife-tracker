About This Project

The Wildlife Sighting Tracker is a web application designed to manage and explore wildlife observation data from across Canada. Built with Express.js, it provides a RESTful API to query sightings by species, habitat, verification status, and more.

Available API Endpoints

GET

/api/sightings

Retrieve all wildlife sighting

GET

/api/sightings/verified

Get only verified sighting

GET

/api/species/species-list

List all unique species names

GET

/api/sightings/habitat/forest

Find all sightings in forest habitats

GET

/api/sightings/search/eagle

Search for eagle sightings

GET

/api/sightings/find-index/moose

Get index position of moose sighting

GET

/api/sightings/recent

Get 3 most recent sightings
